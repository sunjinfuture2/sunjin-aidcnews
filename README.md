# AI Data Center 동향 모니터링

정부기관·공기업 보도자료와 AI데이터센터(AIDC) 관련 뉴스를 한 페이지에 모아 보여주는 대시보드입니다.

## 페이지 구성

- **사이드바(올라온 자료들)**: 공기업 중 오늘 등록된 보도자료를 한눈에 모아 보여줍니다.
- **정부기관 탭**: 과학기술정보통신부, 기후에너지환경부, 산업통상부, 국가인공지능전략위원회 — 자동 파싱 없이 게시판 바로가기 링크만 제공합니다.
- **공기업 탭**: 한국전력공사, 한국수자원공사, 정보통신산업진흥원, 한국지능정보사회진흥원 — 오늘 등록된 글과, 날짜와 무관한 최근 5건("더보기")을 함께 보여줍니다.
- **지자체 관련 기사 탭**: 네이버 뉴스에서 "AI데이터센터/AIDC/AI팩토리" + 국내 지역명이 함께 언급된 기사 최신 20건.
- **국내 기업 관련 기사 탭**: 위 키워드 + 국내 기업명이 함께 언급된 기사 최신 20건.
- **해외 기업 관련 기사 탭**: 위 키워드 + 해외 기업명이 함께 언급된 기사 최신 20건.

## 파일 구성

- `scraper.py`: 각 기관/기업 게시판 및 네이버 뉴스 API에서 데이터를 수집해 `public/index.html`을 생성합니다.
- `data/company_names.txt`: 국내 기업명 목록 (한 줄에 하나씩).
- `data/company_names_overseas.txt`: 해외 기업명 목록 (한 줄에 하나씩).
- `data/region_names.txt`: 국내 지역명(시/도·시/군/구) 목록 (한 줄에 하나씩).
- `data/press_domain_map.txt`: 언론사 도메인 → 표시용 언론사명 매핑 (`도메인,언론사명` 형식, 한 줄에 하나씩).
- `.github/workflows/update.yml`: 하루 여러 차례(00/03/06/09/12/15시 UTC, 즉 KST 09/12/15/18/21/24시) 자동 실행되어
  `public/index.html`을 갱신하고 커밋 후 GitHub Pages로 배포합니다. Actions 탭에서 수동 실행(`workflow_dispatch`)도 가능합니다.

## 환경변수

네이버 뉴스 검색 API를 사용하는 지자체/국내 기업/해외 기업 탭을 위해 아래 두 환경변수가 필요합니다 (GitHub Actions Secrets에 등록):

- `NAVER_CLIENT_ID`
- `NAVER_CLIENT_SECRET`

설정되지 않으면 해당 탭들은 "뉴스 검색에 실패하여 확인하지 못했습니다"로 표시됩니다.

## 데이터 파일 관리

기업명·지역명·언론사 매핑을 추가/수정하고 싶으면 코드를 건드릴 필요 없이 `data/` 폴더의 해당 txt 파일만 편집하면 됩니다. 단, 편집 후에는 반드시 리포지토리에 커밋해야 다음 자동 실행부터 반영됩니다.

## 배포 주소

리포지토리 Settings → Pages에서 처음 한 번 "Source: GitHub Actions"로 설정하면,
`https://<계정>.github.io/<리포지토리>/` 주소로 접속할 수 있습니다.
